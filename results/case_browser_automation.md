# 评估详情：浏览器自动化测试 (Browser Automation)

**任务类别**: 代码与开发运维 (Code DevOps)  
**总体得分**: 1.0 / 1.0（满分）

## 🎯 任务描述

工作区中提供了一个 `shop.html`（一个包含购物车功能的电商产品页面）。
要求 Agent 先阅读理解页面 DOM 结构和 JavaScript 行为，然后编写完整的 Playwright 端到端测试脚本 `test_shop.py`，自动化执行以下多步骤工作流：
1. 打开页面
2. 验证缺货商品按钮处于禁用状态
3. 添加无线鼠标到购物车
4. 添加机械键盘到购物车
5. 验证购物车总价
6. 移除商品并验证更新后的总价
7. 完成结账并验证订单确认信息
